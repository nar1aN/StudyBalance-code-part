# StudyBalance

### About StudyBalance
StudyBalance is a web app for students (school, college, and university) and anyone who wants to manage their time effectively.

Created by nar1aN (Max N.)

[Read more about StudyBalance](https://github.com/nar1aN/StudyBalance-webapp/blob/main/mnechesov_application.md)

---

### About this GitHub repository
This repository contains the source code of the StudyBalance web app.

### Features
- Task planning by day and week
- Workload analysis
- Overload detection
- Smart rescheduling

### Tech Stack
- Backend: Python (FastAPI)
- Frontend: HTML, JS
- Database: PostgreSQL
